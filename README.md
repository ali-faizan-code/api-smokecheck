# API Smoke Check

Python-based API smoke-check validator with pytest coverage.

## Features
- Schema validation
- Post summarization
- JSON report generation
- Response time measurement
- Pytest test suite
- Edge-case and error-path testing

## Files
- `api_validator.py` — main validator logic
- `test_api_validator.py` — pytest test suite

## Usage
Run the validator:

```bash
python api_validator.py

## Run tests
```bash
pytest -v
